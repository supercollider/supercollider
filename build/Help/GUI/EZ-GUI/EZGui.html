EZGui an abstract superclass for EZ widget wrappers


Inherits from: Object


Users will not normally directly create instances of EZLists, but only use it through its subclasses. It provides the basic mechanisms for various EZ widget wrappers.


See also: EZListView, EZPopUpMenu, EZSlider, EZNumber, EZRanger



Accessing Instance and Class Variables

labelPosition

labelPosition_(pos)

Sets the label position. The width and height of the label return to the defaults: 80@this.height or this.width@20;

pos - \left or \top


labelWidth

labelWidth_(width)

Sets/gets the width of the label to width, and adjusts the widget.bounds.. Does nothing if labelPosition=\top;

width - A number;

labelHeight

labelHeight_(height)

Sets/gets the height of the label to height, and adjusts the widget.bounds. Does nothing if labelPosition=\left;

height - A number;
